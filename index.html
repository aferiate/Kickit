Nametag Project

Our first project, the nametag, was designed to teach us the 3D printing process, which starts as either a .3MF or a .STL file. My process was all based on instructions given orally and in the project description file. I started by importing the nametag template file given into Prusa Slicer— the program designed for the Prusa MINI+— and the software scales and places a model of the object onto a representation of the 3D print base. Then, I edited the template name tag and font to change it to be my name and my aesthetic preference. I hit “Slice,” telling the program that the file is finalized and this is what will be sent to print. I then grabbed a USB drive, plugged it into my computer, and dragged the file into the drive. I brought the USB drive to one of the Prusa MINI+ 3D printers and plugged it in. This time, the printer was already ready to print, but it may need to be turned on and/or loaded with filament. The Prusa MINI+ automatically suggested to print the most recently added file to the drive; I hit print and it started doing so. After the first layer finished, the Prusa MINI+ stopped printing and beeped to tell the user it was ready for a color change. I pulled the end of the spool of filament out of the printer, then grabbed a different colored spool and inserted it. The printer asked if I was ready, and upon hitting “continue” I pushed the filament in further as the gear inside loaded it in. The printer tested to see if the new color was working, which it was, so I clicked as such and it went on printing the next color, finishing within a few minutes.

Torture Test

Our second project, the Torture Test, was designed to push the Prusa Mini to its limits. I chose this spider, an intricate design with strange angles and small outshoots. The main constraint of the printer that this ran up against was overhangs. When a part of a design extends outward from its base, with no direct support underneath, it can be very challenging to print; the nozzle ends up extruding plastic straight into the air without it connecting to anything below. The typical rule, though, is any overhang with less than a 45 degree angle to its base is able to support itself. Overall, the details in this print were suitable for both the file and the print, but some hidden features kept it stable, like printing starting at the body and ending at the legs, and having a small flat plane on the top of the body to ensure bed adhesion. Even with all these, there were still many stringy, small filaments protruding from the legs, a result of the hot nozzle traveling far distances across space and leaking instead of traveling more distance across what had been printed and improving the aesthetic product. The file here was originally a .STL file, which only encodes geometry of the 3D model. Once passing through Prusa Slicer, however, the file becomes a .3MF file, which both encodes geometry as well as printer settings, like filament specifications, color changes, supports, cooling rates, etc.

Finger Pen Holder

For this project, the aim was to print some kind of assistive device that has some utility; I chose this finger pen holder. This print proved to be fairly easy- the shape is simple and small- although it needed a plethora of organic supports due to the large overhang in the middle. But after removing the supports, it worked very well. This design, which was created by “Makers Making Change,” a Canadian nonprofit aiming to distribute assistive technology to those who need it, is meant for those with impaired grip strength “like those with arthritis, spinal cord injuries, and muscular dystrophy.” After some testing, I found it to be robust and effective at its task, alleviating grip tension needed to hold a normal pencil tightly. The print documentation was sufficient, (it suggested 90% infill, though 15 was fine) it just neglected to mention the need for supports to mitigate the overhang. My only suggestions for a next prototype would be somehow making both the pen holder side and the finger side adjustable to allow for user preferences of comfort, etc.
